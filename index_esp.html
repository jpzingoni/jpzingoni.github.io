Evolución tickers financieros

El tablero de Evolución tickers financieros fue un desarrollo de modo de trabajo Freelance para un cliente. Con esta idea decidí escalarlo para incorporar mayores indicadores y Tickers. Este proyecto comenzó en enero 2026 y todavía se está trabajando para seguir incoporarando datos.

Contiene información para distintos tickers para poder ver un valor histórico. El mismo utiliza una libreria de Python de Yahoo Finances para obtener los datos. Luego se realizó un proceso de ETL con la liberia de Pandas en Pyrhon para obtener la información limpia y luego realizar el tablero en Power Bi. Se pueden ver datos de precios, volumen y valores contables para estimar un P&L

Estimaciones agrícolas

El tablero de estimaciones agrícolas fue desarrollado para Infodash un proyecto el cual integro donde trabajamos con datos estadísticos y creamos tableros para realizar las visualizaciones. Infodash surge con la idea de procesar los datos para que el usuario la analice de manera propia.

Contiene información para las campañas de Argentina desde 1969. El mismo combina dos bases de datos, una de donde se obtuvo la información relacionada a la estructura principal agricola y otra con las coordenadas de geolocalización de los lugares. Se realizó un proceso de ETL con python para obtener la información limpia y luego realizar el tablero en Power Bi
