Agricultural estimates

The agricultural estimates dashboard was developed for Infodash,a project I'm involved in where we work with statistical data and create dashboards for visualizations. Infodash arises with the idea of processing data so that the user can analyze it in their own way.

It contains information for Argentina's agricultural campaigns since 1969. The dashboard combines two databases, one from which information related to the main agricultural structure was obtained, and another with geolocation coordinates of the places. An ETL process was carried out with Python to obtain clean information and then create the dashboard in Power BI.

Southwest population report

The population report utilizes data from the 2022 Census for Argentina. The information, initially published by INDEC for the national territory, underwent an ETL (Extract, Transform, Load) process to filter and extract details regarding the population evolution specifically for the localities in the southwestern region of Buenos Aires.
