Financial Ticker Evolution

The Financial Ticker Performance dashboard was originally developed as a freelance project for a client. I then decided to scale it to incorporate additional indicators and tickers. Started in January 2026, the project is still under active development.

It covers multiple tickers with historical value analysis. Data is retrieved using the Yahoo Finance Python library, then cleaned through an ETL process with Pandas, and finally visualized in Power BI. The dashboard includes price data, volume, and financial metrics to estimate a P&L.
