Estimaciones agrícolas

El tablero de estimaciones agrícolas fue desarrollado para Infodash un proyecto el cual integro donde trabajamos con datos estadísticos y creamos tableros para realizar las visualizaciones. Infodash surge con la idea de procesar los datos para que el usuario la analice de manera propia.

Contiene información para las campañas de Argentina desde 1969. El mismo combina dos bases de datos, una de donde se obtuvo la información relacionada a la estructura principal agricola y otra con las coordenadas de geolocalización de los lugares. Se realizó un proceso de ETL con python para obtener la información limpia y luego realizar el tablero en Power Bi

Informe de graduados

El informe de graduados utiliza la información de una encuesta que se realiza luego de finalizar la carrera. El tablero permite filtrar por distintas categorias para saber como es el perfil de las personas que egresan. Algunas de las personas graduadas poseen emprendimientos personales y la información permite conocerlos para que en el futuro desde la universidad se puedan aportar herramientas para que los mismos crezcan
